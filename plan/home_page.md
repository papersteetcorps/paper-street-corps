# HOME PAGE


## 1. Visual Style

- Text font: Antonio
- Background (dark mode): Navy blue
- Tab text color (dark, unhovored): Yellow
- Tab text color (dark, hovered): Green
- Regular text color (dark mode): White
- Background (light mode): White
- Tab text color (light, unhovored): Orange
- Tab text color (light, hovered): Green
- Regular text color (light mode): Black
- Current tab: Orange underline
- Other tabs: No underline


## 2. Elements

- Company logo: Paper Street Corps
- Tabs: Home, Typlogy tests, Typology theories, Software tools, About, Team
- Drop down menu: From Typology tests, Typology theories, Software tools
- Theme toggle switch: Dark mode (default), light mode
- Sign-up button: Shaped oval
- Log-in button: Shaped oval
- Labels: (Free/Paid), (Typology/Software)


## 3. Elements Positioning

- Company logo: Header strip
- Tabs: Header strip
- Drop down menu: When hovered over Typology tests, Typology theories, Software tools
- Images: For each card shown as independent list, or as suggestion in menu, and NOT in drop down view.
- Theme toggle switch: Top right corner on head strip
- Sign-up button: Left to theme toggle
- Log-in button: Left to sign-up button


## 3. Functioning

- Clicking directly on any tab lands user to another page where a list is shown with elements that were to appear in drop down menu.
- Otherwise clicking on an element from drop down menu directly lands user to the respective page.


## 4. Landing Interface

- Below header strip: Latest update cards with boxed boundary and respective image on left-side, with two labels in bottom right corner.
- Below update cards: Vertically arranged suggestion cards with boxed boundary and respective image on left-side.
- At last, ethos section is there.


## 5. Important notes

- Update cards must be scalable.
- Suggestion cards must be scalable.
- Tabs must be scalable.
- And create a root file which says which file stores which part/feature.
