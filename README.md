# Paper Street Corps

A minimal personality assessment platform.

## Features

- **MBTI Test** - 16-type personality assessment using neurochemical trait mapping
- **Temperament Test** - Classical four temperaments with blend detection
- **Theory Library** - Educational content on Jungian Typology, Moral Alignment, and Enneagram

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
